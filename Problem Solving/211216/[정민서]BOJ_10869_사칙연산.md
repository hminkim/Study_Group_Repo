# 백준 10869번 사칙연산

[백준 10869번 - 사칙연산](https://www.acmicpc.net/problem/10869)  
  
  
### 소스 코드

```py
digit1, digit2 = input().split()

print(int(digit1) + int(digit2))
print(int(digit1) - int(digit2))
print(int(digit1) * int(digit2))
print(int(digit1) // int(digit2))
print(int(digit1) % int(digit2))

```
