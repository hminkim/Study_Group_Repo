# 백준 15717 떡파이어

[백준 15717 - 떡파이어](https://www.acmicpc.net/problem/15717)  
  
  
### 소스 코드

```c
#include <stdio.h>

long long mod(long long a, long long b, long long mod) 
{
	long long sum = 1;

	while (b) {
		if (b % 2) sum = sum * a % mod;
		a = a * a % mod;
		b = b/2;
	}

	return sum;
}

int main() 
{
	long long n;
	scanf("%lld", &n);
    
	if (n == 0) 
        printf("%d", 1);
	else 
        printf("%lld", mod(2, n - 1, 1000000007));
}

```
